Dashboard em Python para análise de faturamento, metas, clientes e indicadores estratégicos conectados ao ERP Data7.

---

## Objetivo

Transformar dados operacionais em informações gerenciais para apoiar decisões rápidas e melhorar performance financeira.

---

## Principais Funcionalidades

✅ Faturamento diário, semanal e mensal  
✅ Evolução de receita por período  
✅ Meta vs realizado  
✅ Ranking de clientes  
✅ Ticket médio  
✅ Indicadores de produtividade  
✅ Tendências e comparativos  
✅ Visualização executiva em dashboard

---

## Resultados Reais

- Mais velocidade na tomada de decisão  
- Visão clara de metas e desempenho  
- Menos dependência de planilhas manuais  
- Melhor acompanhamento comercial e financeiro  
- Gestão baseada em dados

---

## Tecnologias Utilizadas

- Python
- Streamlit
- SQL
- PostgreSQL
- Pandas
- Plotly
